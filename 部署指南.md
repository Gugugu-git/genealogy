# 泸县大堰胡氏宗谱管理系统 - 部署指南

## 📋 部署方式总览

本项目支持以下三种部署方式：

1. **Web静态部署** - 部署到云平台（Vercel/Netlify/GitHub Pages）
2. **传统服务器部署** - 部署到自己的服务器（Nginx/Apache）
3. **桌面应用打包** - 打包为Windows桌面应用（推荐）

---

## 方案一：Web静态部署（推荐新手）

### 优点
- ✅ 免费托管
- ✅ 自动HTTPS
- ✅ 自动部署
- ✅ 访问速度快

### ⚠️ 重要提示
Web版本需要修改数据存储方式，将本地JSON文件改为localStorage

### 1. 部署到 Vercel（推荐）

#### 步骤：
1. 安装 Vercel CLI
```bash
npm install -g vercel
```

2. 登录 Vercel
```bash
vercel login
```

3. 部署项目
```bash
vercel
```

4. 按提示操作即可完成部署

#### 或者通过 GitHub 自动部署：
1. 将代码推送到 GitHub
2. 访问 [vercel.com](https://vercel.com)
3. 导入 GitHub 仓库
4. 自动部署完成

### 2. 部署到 Netlify

#### 步骤：
1. 安装 Netlify CLI
```bash
npm install -g netlify-cli
```

2. 构建项目
```bash
npm run build
```

3. 部署
```bash
netlify deploy --prod --dir=dist
```

#### 或者通过网页部署：
1. 访问 [netlify.com](https://netlify.com)
2. 拖拽 `dist` 文件夹到页面
3. 自动部署完成

### 3. 部署到 GitHub Pages

#### 步骤：
1. 修改 `vite.config.js`
```javascript
export default defineConfig({
  plugins: [react()],
  base: '/你的仓库名/',  // 改为你的GitHub仓库名
  server: {
    port: 5173
  }
});
```

2. 安装 gh-pages
```bash
npm install --save-dev gh-pages
```

3. 在 `package.json` 中添加脚本
```json
{
  "scripts": {
    "deploy": "npm run build && gh-pages -d dist"
  }
}
```

4. 部署
```bash
npm run deploy
```

## 方案二：传统服务器部署

### 1. 构建生产版本

```bash
npm run build
```

构建后会生成 `dist` 文件夹，包含所有静态文件。

### 2. 上传到服务器

使用 FTP/SFTP 工具将 `dist` 文件夹上传到服务器

### 3. Nginx 配置示例

```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /var/www/genealogy/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    # 启用gzip压缩
    gzip on;
    gzip_types text/css application/javascript application/json;
}
```

### 4. Apache 配置示例

在 `dist` 文件夹中创建 `.htaccess` 文件：

```apache
<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  RewriteRule ^index\.html$ - [L]
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteRule . /index.html [L]
</IfModule>
```

## 方案三：桌面应用打包（推荐）

### 优点
- ✅ 保持原有功能
- ✅ 本地数据存储
- ✅ 无需网络
- ✅ 更安全

### 步骤：

1. 安装 Electron 依赖
```bash
npm install --save-dev electron electron-builder concurrently wait-on
```

2. 打包 Windows 应用
```bash
npm run electron:build
```

3. 打包完成后，在 `dist-electron` 文件夹中找到安装包

### 分发方式：
- 将安装包发送给家族成员
- 双击安装即可使用
- 数据完全本地存储

## 🔧 Web版本数据存储修改

如果选择Web部署，需要修改数据存储方式：

### 修改方案：使用 localStorage

1. 数据存储在浏览器本地
2. 每个用户独立数据
3. 支持导入/导出功能

### 优点：
- ✅ 无需后端服务器
- ✅ 数据完全本地化
- ✅ 支持离线使用

## ❓ 常见问题

### Q: Web版本数据会丢失吗？
A: localStorage 数据存储在浏览器中，清除浏览器数据会丢失。建议定期导出备份。

### Q: 桌面应用可以在其他电脑使用吗？
A: 可以！打包后的安装包可以在任何Windows电脑上安装使用。

### Q: 如何更新数据？
A: 桌面应用直接编辑JSON文件；Web版本使用导入/导出功能。
