# 泸县大堰胡氏宗谱信息管理系统

一个纯本地存储的族谱管理工具，无需云端依赖，数据文件保存在本地电脑。

## 功能特性

### 核心模块
- **族谱总览**：展示宗谱完整目录，点击跳转对应模块
- **世系展示**：树形结构+文字列表双模式，支持搜索和筛选
- **字辈查询**：完整字辈展示，点击字辈查看对应人物
- **凡例规则**：完整展示7条凡例，作为族谱更新依据
- **谱序后跋**：展示原文，支持应用内编辑
- **数据管理**：可视化编辑、备份管理、双向同步

### 数据管理
- 本地JSON格式存储
- 应用内可视化编辑+本地文件手动编辑双模式
- 自动备份最近10个版本
- 1-6世核心世系删除保护
- 修改日志记录

## 技术栈

- **Electron**：桌面应用框架
- **React**：前端UI框架
- **Vite**：构建工具
- **React Router**：路由管理

## 安装和运行

### 前置要求
- Node.js 16+ 
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 开发模式运行
```bash
npm run electron:dev
```

### 构建生产版本
```bash
npm run electron:build
```

### 仅运行Web版本（不使用Electron）
```bash
npm run dev
```

## 使用说明

1. **首次打开**：系统会显示操作指引，帮助您快速上手
2. **查看世系**：进入「世系展示」模块，可以选择树形或列表模式
3. **编辑数据**：进入「数据管理」模块，可以新增人物、编辑信息
4. **备份恢复**：系统自动备份，可在「数据管理-备份管理」中恢复
5. **文件同步**：支持从本地JSON文件同步数据到应用内

## 数据格式说明

数据文件采用JSON格式，主要包含以下字段：
- `metadata`：元数据信息
- `catalog`：目录结构
- `genealogy`：世系人物数据
- `generation`：字辈信息
- `rules`：凡例规则
- `preface` / `postscript`：谱序/后跋
- `backupHistory` / `changeLog`：备份/修改日志

## 注意事项

- 所有数据保存在本地，请勿删除数据文件
- 1-6世核心世系人物受删除保护，无法删除
- 建议定期备份数据文件
- 修改前可先创建备份
